Overview

Soyatec Tooling Project provided toolings for developing GEF Graphical Editors, Eclipse Views based on Tree and externalization for Chinese.

Toolings

  • Toolings for GEF Graphical Editor
    • Diagram interchange model based on EMF.
    • Graphical editor with ToolBar.
  • Translation eclipse workbench to Chinese
    • Chinese Language pack for eclipse workbench plug-ins.
  • Tooling for developing Views
    • View with TreeViewer.
    • PropertySheetPage with Form Section.
  • XML tooling based on EMF (EXML)
    • Ease loading and saving XML with EMF.
  • RichText HTML Editor
    • HTML WYSIWYG editor.
